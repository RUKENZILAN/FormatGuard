Runs entirely in your browser — nothing is sent anywhere. Every conversion passes through five checks before it's rendered: size cap, structural depth, node count, prototype-pollution keys (__proto__, constructor, prototype), and an expansion-ratio check that catches YAML "billion laughs" anchor bombs. YAML is parsed with a JSON-only schema, so executable or custom tags (e.g. !!js/function) are rejected outright. Output is written via textContent, never innerHTML, so a malicious payload can't execute as markup.
